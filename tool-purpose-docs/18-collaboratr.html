What It Is

The Creative Director Chat

CoLABoratr is a two-panel chat interface where Claude loads your full project context — the approved script, beat map, research findings, content angle, and creative package — and becomes your creative director for the session. You ask it anything about the project. It answers with the whole picture in mind.

It's not a sequential pipeline step. It's the smart collaborator you pull in when you're stuck, when you're thinking something through, or when you want a second opinion before committing to a direction. It doesn't produce a file or a database record. The output is clarity. You leave knowing what to do next.

Use it to think through specific shots for a new video — what moments would actually tell the story, which ones could be cut, what the camera needed to capture versus what the edit would carry. That kind of thinking used to happen in your head, alone, or in a production meeting. CoLABoratr gives you a room to think out loud in, with someone who already knows the project.

How It Works

The Workflow

1
Open Lab from the nav — select a project
CoLABoratr lives at /lab.html. Branded as "🧪 LAB" in the nav. Select any active project to load its context.
2
Full project context loads automatically
Context pulled via /api/lab/context/{project_id}: approved script, beat map, research findings, content angle, creative package, creator voice profile. Claude's system prompt is built from all of it.
3
Chat interface opens — type any creative question
No form fields, no modes, no setup. Just a chat box. The sidebar shows project info: beats, angle, script excerpt — so you always have the map in front of you.
4
Claude responds as creative director with full project awareness
Responses stream via /api/lab/chat (SSE). Claude knows the voice, the angle, the structure, the beats. It doesn't give you generic advice — it gives you answers about this project.
5
Conversation held in memory for the session only
Chat history is not persisted. CoLABoratr is a thinking space, not a record. When you close the tab, the conversation ends. That's intentional — it keeps the space clean and low-stakes.
6
Walk away knowing what to do — go do it in the right tool
The decisions you made in CoLABoratr get implemented wherever they belong: DirectΩr, EditΩr, WritΩr, ShootDay. CoLABoratr is the thinking. The other tools are the doing.

When To Use It

Pull It In When You're Stuck

  • Before a shoot when you're not sure what shots will actually tell the story
  • When the script feels off and you can't identify why
  • When you want to pressure-test a hook or angle before committing to it
  • When you've gone off-script during a shoot and need to think through what you've got
  • When you're stuck mid-edit and need a second opinion on structure or pacing
  • When something in the pipeline doesn't feel right and you can't name it yet
