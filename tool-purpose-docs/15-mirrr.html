What It Is

The Channel Mirror

MirrΩr is where the pipeline looks at itself. Every video you make generates a signal — views, watch time, CTR, comments, community response — and that signal contains information about what to make next. MirrΩr collects that signal, runs it through Claude, and turns it into creative intelligence for the next video. It's the difference between guessing what your audience wants and knowing.

The channel mirror metaphor is intentional. You look into it and see your content universe reflected back — which angles are working, which formats your audience returns for, where retention collapses, what the comments are actually asking for. Then that reflection becomes the starting point for Id8Ωr's next ideation session. The pipeline becomes a loop, not a line.

MirrΩr also holds your YouTube video link connection, pulling the video ID from the URL and storing it against the project so downstream analytics can reference it. Every project that passes through MirrΩr is permanently tethered to its real-world performance data.

How It Works

Step by Step

1
Load Project
Project loads with its full pipeline context. YouTube URL field pre-fills if previously linked.
2
Paste YouTube URL — Link or Link & Sync
"Link" or "Link & Sync" saves youtube_url and the extracted youtube_video_id to the project via PATCH /api/projects/:id/meta — permanently tethering the project to its published video.
3
Pull Performance Data
Where available, performance metrics are pulled: views, retention curve, CTR, average view duration, top comments. YouTube API integration in active development.
4
Claude Analyzes the Signal
Claude receives performance data alongside the original creative brief. Analysis covers: strongest retention moment, drop-off patterns, comment appetite signals, and angle performance compared against the channel's full history.
5
Coaching Insights Surface
Insights display: which angle outperformed, what format the audience returned for, where the hook landed or missed. Content Universe map shows all content angles and their relative performance in one visual.
6
Strongest Signals Flagged as Seeds → Passed to Id8Ωr
Top insights flagged as ideation seeds. When the creator opens Id8Ωr for the next video, MirrΩr's intelligence is waiting — a recommended starting direction built from what actually worked, not guesswork.
Build Status
YouTube URL linking and video ID extraction are live. Full YouTube Analytics API pull, coaching insights display, and Content Universe map are in active development. Current build stores and links the YouTube URL — analytics intelligence layers in next.
