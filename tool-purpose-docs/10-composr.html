What It Is

The Film Scorer

Music is the thing most solo creators skip because hiring a composer is expensive and browsing stock libraries is a black hole. ComposΩr eliminates both problems. It reads what's happening emotionally in each scene of your video — the tension, the warmth, the payoff — and writes music that actually fits those moments instead of music that's merely inoffensive.

It uses Claude to analyze your transcript and script, identify the emotional arc scene by scene, and generate precise prompts for Suno (an AI music generator). Those prompts go to Suno, tracks come back, you preview them in the browser, pick one per scene, and hit Push to DaVinci. The audio lands on the 04_AUDIO track in your Resolve timeline at the exact timestamps — no manual dragging, no track naming, no export-and-import loop.

Two modes: Full Mode calls the Suno API automatically and returns generated tracks for you to choose from. Prompt Mode is the fallback — Claude writes the prompts, you paste them at suno.com yourself, then upload the audio files back. Either way, the music ends up in Resolve.

How It Works

Step by Step

1
Select Project
The approved transcript and script load automatically from the project record. Both are available as context for scene analysis.
2
Analyze Scenes
Claude reads the full transcript and script, identifies the emotional arc, and segments the video into scenes. Each scene gets a label (e.g. "opening tension", "revelation moment", "warm resolution") and an emotional tone tag stored in composor_scenes.
3
Generate Music Prompts
Claude writes a Suno-optimized prompt for each scene — genre, tempo, instrumentation, mood, energy level. Prompts are crafted to produce music that fits the emotional beat, not just the topic.
4
Suno Generates Tracks (Full Mode)
Suno API is called for each scene. Tracks return as audio files, previewed directly in the browser. Full Mode requires Suno API configured. Prompt Mode skips this step — creator pastes prompts at suno.com manually.
5
Creator Selects a Track Per Scene
Each scene shows its candidate tracks. One click selects the keeper. Selection stored in composor_tracks with scene_id, audio_path, and selected flag.
6
Push to DaVinci
Python script calls the DaVinci Resolve Studio scripting API on port 9237. Selected audio files are placed on the 04_AUDIO track at the correct timestamps. Resolve must be running. stdout = JSON confirmation, stderr = logs.
Technical Requirement
DaVinci Resolve Studio 20.3.2.9 must be running before any Push to DaVinci action. Python scripting API runs on port 9237, Local mode, Windows only. All Python scripts use callable() guards for Resolve 20 API compatibility.
What It Creates

Data & Outputs

Data & Records
composor_scenes rows — scene_index, scene_label, emotional_tone, suno_prompt per scene
composor_tracks rows — scene_id, audio_path, selected flag for every generated track
project record updated — music status flagged when push to DaVinci completes
Files & Outputs
Audio files — saved to public/music/ per scene
DaVinci timeline modification — 04_AUDIO track populated via Python scripting API with correct timestamps
Suno prompts — exportable as text (Prompt Mode) for manual paste at suno.com
Valuable Final Product

What You Walk Away With

A fully scored video edit — AI-generated music placed into DaVinci with the emotional arc matched to sound. The 04_AUDIO track is populated, the timestamps align, and you didn't hire a composer, browse a library, or drag a single audio clip. The video sounds intentional because the music was written for it.
