What It Is

The Platform Voice Translator

Each platform has its own personality and its own audience expectations. What works on TikTok will sound wrong on Facebook. What Instagram rewards is not what YouTube Shorts ranks. CaptionΩr knows the difference. It takes each approved clip and writes native captions for all five platforms at once — not one caption copy-pasted across five places, but five different captions written for five different contexts.

Every caption is written in the creator's voice, informed by the selected content package from PackageΩr. The hook, angle, and tone are baked into the system prompt before Claude writes a single word, so the captions stay consistent with the video's creative direction without the creator having to manually inject that context each time.

The creator reviews, edits any individual caption, or hits Approve All. Done. The entire short-form posting week — every clip, every platform — written in one generation.

TikTok
Under 300 chars. One-line scroll-stopper. Hook in the first 3 words.

Conversational paragraphs. Community discussion opener.

How It Works

Step by Step

1
Load Project
Approved clips from ClipsΩr load automatically. Selected package from PackageΩr loads as the tone anchor — title, hook, and angle inject into the system prompt before any caption is written.
2
Add Optional Direction Note
Free-text creator input: tone adjustments, words to avoid, emphasis points. Applied across all captions in this generation run.
3
Claude Writes 5 Captions Per Clip
All approved clips processed simultaneously. Platform rules enforced per caption. Token budget set to 8192 — required for 6 clips × 5 platforms in one generation. Each caption is written in the creator's voice from creator-profile.json.
4
Creator Reviews
Every caption is individually editable inline. "Approve All" button approves all captions across all clips and platforms in one click. Individual approvals available if the creator wants to review platform by platform.
5
Approved Captions Stored
Written to captions table: project_id, clip_id, platform, caption_text, approved. MailΩr reads caption context to inform email tone for the broadcast sequence.
