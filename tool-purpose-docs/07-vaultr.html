What It Is

The Footage Brain

VaultΩr is a living database of every piece of video the creator has ever shot. Not a folder. Not a manually-tagged library. A database — where every clip has been seen by Claude Vision, classified by shot type, described in plain language, and linked to the project it came from.

It works by watching a folder. DaVinci Resolve exports proxy files into D:\kre8r\intake — VaultΩr's watcher detects the new file, immediately kicks off the intake pipeline, and by the time the creator switches windows, the clip is classified, thumbnailed, and in the database. No drag-and-drop. No tagging screen. No decisions required.

B-roll from three years ago is findable. Talking-head footage from last Tuesday is linked to its project. A voice analysis button on completed-video cards feeds the WritΩr voice library — so the system gets more accurate about the creator's voice the more content runs through it.

How It Works

Step by Step

BRAW Proxy Workflow: Blackmagic cameras shoot .braw files — VaultΩr creates the BRAW record first. When DaVinci exports the proxy, it links back via the _proxy.mp4 naming convention (findBrawByBasename). No manual linking required.
1
DaVinci Exports Proxy
After the shoot, DaVinci Resolve exports H.264 proxy files from the BRAW originals. Proxy lands in D:\kre8r\intake. The originals stay on the camera SSD or D:\ archive — VaultΩr never touches them.
2
Watcher Detects New File
VaultΩr's folder watcher fires the moment a new .mp4 appears in the intake path. No polling delay. The intake pipeline starts immediately.
3
ffmpeg Extracts Metadata
Duration, resolution, codec, frame rate, and a thumbnail image are extracted via ffprobe and ffmpeg. All stored in the DB. Thumbnail saved to public/thumbnails/.
4
Claude Vision Classifies Shot Type
Thumbnail is sent to Claude Vision. Shot type classified as: talking-head, b-roll, action, dialogue, completed-video, or unusable. No manual tagging required.
5
Record Written to Database
All metadata, classification, filepath, and thumbnail stored in the footage table. If the file matches a BRAW record via _proxy.mp4 naming, the records are linked automatically.
6
Creator Reviews + Adjusts
Creator can override classification, add a description, link a clip to a specific project, or flag a clip as unusable — directly from the VaultΩr card UI. These manual overrides are stored and respected downstream.
7
Voice Analysis (Completed Videos)
Completed-video cards show a "Analyze Voice" button. Runs Whisper transcription + Claude voice profiling. Output feeds the WritΩr voice library — the system learns from real delivered content.
