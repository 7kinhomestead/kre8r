What It Is

The Content Strategist

By the time a video reaches PackageΩr, a lot of creative work is done — the script exists, the transcript exists, the viral clips and hooks exist. PackageΩr's job is to take all of that and answer the distribution question: what's the best angle to lead with, and why, for this specific video and this specific audience?

It generates 5 distinct content packages. Not 5 variations of the same idea — 5 genuinely different angles, each a complete creative direction. Every package includes a YouTube title, a platform hook, a rationale explaining why that angle works for the channel's audience, a thumbnail concept, and a full YouTube description with the creator's real social links and community CTAs already populated.

The creator picks one. That choice — called Gate A — locks the creative direction. Every tool downstream (CaptionΩr, MailΩr) reads from the selected package, so the title, hook, and angle stay consistent from YouTube to TikTok to the email list without rewriting anything.

How It Works

Step by Step

1
Load Project
Approved clips from ClipsΩr, the approved script, and the full transcript all load as context. ClipsΩr hooks are available to Claude as starting material for package angles.
2
Select Goal and Angles
Creator picks a distribution goal (Grow audience / Drive community / Evergreen / Viral) and one or more content angles (financial, system, show format, how-to, mistakes, lifestyle, viral). These shape which kinds of packages Claude generates.
3
Optional Creator Direction Note
Free-text field: "lead with the cost number", "avoid jargon", "push the inner circle CTA hard". Claude reads this note and applies it across all 5 packages.
4
Claude Generates 5 Packages
Each package: YouTube title, platform hook, rationale (why this angle for this audience), thumbnail concept, full YouTube description with social links and community tier CTAs. Pulled from creator-profile.json — never hardcoded. Packages are genuinely distinct angles, not rewrites of each other.
5
Creator Selects One Package (Gate A)
One click selects the active package. The selected package is stored as the project's creative direction. This is the approval gate — no downstream distribution tool runs without a selected package.
6
Selected Package Stored
Written to packages table with is_selected flag. projects record updated with selected_package_id. CaptionΩr and MailΩr read from this record to anchor tone.

Valuable Final Product

What You Walk Away With

One selected content package — the creative direction the entire distribution layer runs on. Title, hook, angle, thumbnail concept, and YouTube description locked. Every tool downstream stays in lane because the lane was defined here. Without this, every platform gets a slightly different message. With it, every platform amplifies the same one.
