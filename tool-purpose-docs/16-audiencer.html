What It Is

The Audience Lens

Writing emails and community posts without knowing who's in the community is writing blind. AudiencΩr fixes that. It connects directly to the Kajabi community database via OAuth2 and pulls all contacts — their names, email addresses, tags, active offers, and membership status — in real time, with no export, no CSV, no spreadsheet.

The community has three tiers, each with a different relationship to the creator and a different expectation from the content they receive. Seeing those numbers — how many in the free tier, the paid tier, and the inner circle — before you write an email changes how you write it. AudiencΩr is planned to live as a sidebar panel inside MailΩr so this data is visible at exactly the moment it's needed.

It's a data source, not a pipeline step. It doesn't create records or files. It reads. But reading with accuracy is the foundation of every targeting decision downstream.

How It Works

Step by Step

1
OAuth2 Connect to Kajabi
Client credentials flow — no user login required. Kajabi Public API OAuth2 connected at the application level. Token managed server-side.
2
Pull All Contacts
GET /contacts — Kajabi returns all contacts at once (no pagination). Name, email, tags, active offers, and membership status per contact displayed in the UI.
3
View Tags and Offers
All tags and active offers surfaced per contact. Tag filter available — see a specific segment. Note: Kajabi 500s on filtered tag requests; full list works correctly.
4
Broadcast-Tag SSE
Real-time server-sent event stream shows live tag operations as they happen — useful for monitoring active campaigns or community events in progress.
5
Inform MailΩr and AutomatΩr
Audience data informs tier selection in MailΩr (who gets which email) and targeting logic in AutomatΩr (which segment to send to). AudiencΩr itself takes no action — it surfaces the data, then gets out of the way.
Known Issue
Kajabi returns a 500 error on filtered tag requests. Tag filter in the UI is non-functional. Full contact list with all tags loads correctly. Fix requires investigation of Kajabi API behavior on tag-filtered endpoints.

Valuable Final Product

When you know the inner circle is 47 people and the free tier is 2,300, you write the Day 7 email differently. That clarity is what AudiencΩr provides.
