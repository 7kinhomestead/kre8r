⚙️  Platform Setup — add API credentials to .env to unlock connections

YouTube (Google)

  1. Go to console.cloud.google.com → New Project → Enable YouTube Data API v3
  2. APIs & Services → Credentials → Create OAuth 2.0 Client ID (Web application)
  3. Add Authorized redirect URIs: http://localhost:3000/api/postor/auth/youtube/callback
  4. Add to your .env:
    GOOGLE_CLIENT_ID=your-client-id
    GOOGLE_CLIENT_SECRET=your-client-secret
  5. Restart server → click Connect YouTube

Instagram + Facebook (Meta)

  1. Go to developers.facebook.com → Create App → Business type
  2. Add products: Facebook Login + Instagram Basic Display
  3. App Settings → Basic → get App ID & Secret
  4. Facebook Login → Settings → add Valid OAuth Redirect URI: http://localhost:3000/api/postor/auth/meta/callback
  5. Add permissions: instagram_basic, instagram_content_publish, pages_manage_posts, pages_read_engagement, pages_show_list, publish_video
  6. Add to your .env:
    META_APP_ID=your-app-id
    META_APP_SECRET=your-app-secret
  7. Restart server → click Connect Meta (Instagram + Facebook)
  8. Note: In Development mode you can only post to accounts you add as Test Users. Submit for Advanced Access before going live.
